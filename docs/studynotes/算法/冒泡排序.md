## 冒泡排序

### 一、标准版

```java
public static void main(String[] args) {
    int[] arr = {7, 3, 2, 5, 6, 4};

    for (int i = 0; i < arr.length - 1; i++) {
        for (int j = 0; j < arr.length - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交换两个元素的位置
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
    System.out.println(Arrays.toString(arr));
}
```

思路：

1. 两层循环；
2. 外层控制整个循环的轮数；

3. 内层负责比较并交换两个元素的位置，把大的元素放后面，第一轮结束后，能把最大值移到末尾。

### 二、优化版

```java
public static void main(String[] args) {
    int[] arr = {7, 3, 2, 5, 6, 4};

    for (int i = 0; i < arr.length - 1; i++) {
        // 标志位，表示这次遍历是否发生过交换
        boolean swapped = false;
        for (int j = 0; j < arr.length - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                // 设置标志位为true，表示发生过交换
                swapped = true; 
            }
        }
        // 如果这次遍历没有发生过交换，说明数组已经排序完成
        if (!swapped) {
            break;
        }
    }
    System.out.println(Arrays.toString(arr));
}
```

 这个版本添加了一个标志位来优化冒泡排序，当某一次遍历中没有发生交换时，说明数组已经排序完成，可以直接结束排序。 

### 三、双向冒泡排序（鸡尾酒排序）

```java
public static void main(String[] args) {
    int[] arr = {7, 3, 2, 5, 6, 4};

    boolean swapped;
    for (int i = 0; i < arr.length - 1; i++) {
        swapped = false;
        // 向前冒泡，比较相邻两个元素的大小并交换
        for (int j = 0; j < arr.length - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = true;
            }
        }

        // 如果这一轮没有发生交换，说明数组已经排序完成
        if (!swapped) {
            break;
        }

        swapped = false;
        // 向后冒泡，比较相邻两个元素的大小并交换
        for (int j = arr.length - i - 1; j > 0; j--) {
            if (arr[j] < arr[j - 1]) {
                int temp = arr[j];
                arr[j] = arr[j - 1];
                arr[j - 1] = temp;
                swapped = true;
            }
        }
        // 如果这一轮没有发生交换，说明数组已经排序完成
        if (!swapped) {
            break;
        }
    }
    System.out.println(Arrays.toString(arr));
}
```

双向冒泡排序（也被称为鸡尾酒排序）在每一轮中，既比较前一个元素和后一个元素的大小进行交换，也会比较后一个元素和前一个元素的大小进行交换，这样可以减少排序的轮数。
